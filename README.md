<h1 align="center">🛡️ Shieldhood</h1>

<p align="center">
  <strong>Advanced AI Security Layer for Bankr.bot on Robinhood Chain</strong><br>
  Real-time protection against prompt injection, jailbreaks, and malicious commands for autonomous DeFi agents.
</p>

---

### ✨ Key Features

- Multi-layer injection detection (Base64, Hex, ROT-N, Morse, invisible Unicode, Zalgo, high-entropy, etc.)
- Automatic payload decoding & re-scanning
- Human confirmation gate for high-risk actions
- Spending policy engine (daily & per-transaction limits)
- Address allowlist system
- Lightweight & production-ready (pure Python)
