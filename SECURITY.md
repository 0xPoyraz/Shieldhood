# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 2.0.x   | ✅ Yes             |
| < 2.0   | ❌ No              |

## Reporting a Vulnerability

If you discover a security vulnerability in Shieldhood, please report it responsibly.

**Please do not** report security vulnerabilities through public GitHub issues.

### How to Report

Send an email to: **0xPoyraz@gmail.com**

Include the following information:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Shieldhood version affected
- Any suggested fixes (if known)

### Response Time

- Acknowledgment: within 48 hours
- Detailed response: within 5 business days

## Scope

This policy covers:
- The core `shieldhood` Python package
- Detection logic and scanning engine
- Integration with Bankr.bot

## Disclosure Policy

We follow responsible disclosure:
- We will confirm receipt of your report
- We will work with you to fix the issue
- We will credit you in the release notes (if desired)

Thank you for helping keep Shieldhood secure. 🛡️
